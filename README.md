# 便利店信息管理系统 java351

## 项目概述

本项目是一个基于Springboot+Vue的前后端分离的便利店信息管理系统。该系统遵循MVC三层体系结构，使用Spring、SpringBoot、MybatisPlus、Vue和Maven等技术栈进行开发。系统使用MySQL数据库作为数据存储，实现了基于B/S结构的Web应用。用户界面简洁，操作方便。

## 技术栈

- 后端：Spring、SpringBoot、MybatisPlus
- 前端：Vue
- 构建工具：Maven
- 数据库：MySQL

## 功能模块

- 商品管理
- 库存管理
- 订单管理
- 用户管理
- 报告生成

## 系统角色

- 系统管理员
- 店员
- 顾客

## 运行环境

- 后端：JDK 1.8 或更高版本
- 前端：现代浏览器
- 数据库：MySQL 5.7 或更高版本

## 部署步骤

1. 安装并配置好Java环境
2. 安装MySQL数据库，并导入数据
3. 使用Maven构建项目
4. 部署前端代码到Web服务器
5. 运行后端服务

## 目录结构

```
|-- 项目名称
    |-- backend                 # 后端代码目录
    |-- frontend                # 前端代码目录
    |-- db                      # 数据库脚本目录
    |-- doc                     # 项目文档目录
```
